## Program 2 - 使用單色 LED 實作呼吸燈 (%)

### 作業說明

使用板子右下角的單色 LED ，實作一顆呼吸燈。

### 作業需求

1. 使用一顆 LED 實作產生漸強漸弱的呼吸燈即可。

2. 呼吸燈長[這樣](https://www.youtube.com/watch?v=Z6tbQ0HNmag)

3. 可選擇使用IP -> Block Design的方式，也可以純寫 Verilog code 來實作。
